flutter pub run flutter_launcher_icons:main -> to create icons main icon in assets\icons\ctrainer_master_1024.png

flutter pub run flutter_native_splash:create -> to create splash screen

flutter build appbundle -> generate appbundle

For the android key store it must be added:

android/key.properties
  storePassword=YOUR_STORE_PASSWORD
  keyPassword=YOUR_KEY_PASSWORD
  keyAlias=YOUT_ALIAS
  storeFile=key.jks
